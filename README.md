# Solar In House ☀️

Сучасний вебсайт для компанії Solar In House, яка займається проєктуванням, встановленням та обслуговуванням сонячних електростанцій в м. Києві та Київській області.

## 🚀 Можливості

- Адаптивний дизайн
- Сучасний інтерфейс
- Форма консультації
- Блок послуг
- Портфоліо проєктів
- Контактна інформація
- Оптимізована продуктивність

## 🛠 Технології

- HTML5
- CSS3
- JavaScript
- Font Awesome
